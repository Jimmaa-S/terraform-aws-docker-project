# Terraform AWS Docker Project

## Project Overview
This project demonstrates how to provision AWS infrastructure using: 
**Terraform** and deploy a **Dockerized application on an EC2 instance**. 
It showcases Infrastructure as Code (IaC), cloud automation, and 
containerizationâ€”key skills for deploying **data science applications, APIs, and dashboards** in production.

## Architecture
- AWS EC2 instance (Amazon Linux)
- Custom VPC with public subnet
- Security Group allowing HTTP access
- Docker installed automatically on EC2
- Docker container deployed at instance startup
